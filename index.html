2. Co je MůjAsistent

MůjAsistent je aplikace pro správu a publikaci obsahu na sociálních sítích (Facebook, Instagram) prostřednictvím Meta Graph API. Aplikace umožňuje správcům Facebook stránek a Instagram účtů vytvářet a publikovat příspěvky.

3. Jaké údaje zpracováváme

Aplikace zpracovává pouze údaje nezbytné pro svou funkci:

  • Přístupové tokeny k Facebook stránkám a Instagram účtům (uloženy bezpečně, nikdy nesdíleny)
  • ID Facebook stránek a Instagram Business účtů
  • Obsah příspěvků (text, obrázky) zadaný správcem k publikaci
  • Základní metadata stránek (název, ID) nutná pro fungování API

Aplikace nezpracovává osobní údaje fanoušků ani návštěvníků spravovaných stránek.

4. Účel zpracování

Údaje jsou zpracovávány výhradně za účelem publikace obsahu na Facebook stránkách a Instagram účtech dle pokynů správce. Žádné údaje nejsou prodávány, pronajímány ani sdíleny s třetími stranami.

5. Meta platformy

Aplikace používá Meta Graph API v souladu s Meta Platform Policy a Meta Privacy Policy. Přístup k datům je omezen na oprávnění schválená Meta App Review procesem.

6. Zabezpečení

Přístupové tokeny jsou uloženy bezpečně a nikdy nejsou sdíleny s třetími stranami. Aplikace používá výhradně šifrovaná HTTPS spojení pro komunikaci s Meta API.

Privacy Policy

Last updated: February 2026

1. Operator

The operator of the MůjAsistent application is:
Jakub Žemlička, sole trader (OSVČ)
E-mail: zemlicka.jakub@gmail.com

2. What is MůjAsistent

MůjAsistent is an application for managing and publishing content on social media platforms (Facebook, Instagram) via the Meta Graph API. The application enables Facebook Page and Instagram account administrators to create and publish posts.

3. Data we process

The application processes only data necessary for its operation:

  • Access tokens for Facebook Pages and Instagram accounts (stored securely, never shared)
  • Facebook Page IDs and Instagram Business Account IDs
  • Post content (text, images) provided by the administrator for publication
  • Basic page metadata (name, ID) required for API functionality

The application does not process personal data of fans or visitors of managed pages.

4. Purpose of processing

Data is processed solely for the purpose of publishing content on Facebook Pages and Instagram accounts as directed by the administrator. No data is sold, rented, or shared with third parties.

5. Meta platforms

The application uses the Meta Graph API in accordance with the Meta Platform Policy and Meta Privacy Policy. Data access is limited to permissions approved through the Meta App Review process.

6. Security

Access tokens are stored securely and are never shared with third parties. The application uses only encrypted HTTPS connections for communication with the Meta API.

  • Access your personal data
  • Rectification of inaccurate data
  • Erasure of data
  • Withdrawal of consent to processing

To exercise these rights, contact us at: zemlicka.jakub@gmail.com
